# project2

## This project is a comic book tracker app. the idea is that you can see when the next issue of the comic books you are currently follow is coming out

### tech used
- express.js
- node.js
- mongodb
- mongoose
- JavaScript
- html
- css 
- Marvel API 

### Features
- create user
- add comics 
- tracks comics release dates via Marvel API
